This folder contain the plugin for the combined part of the RoVi final project

How to compile the project:
1) Create build folder
2) Open build folder in cmd
3) Write "cmake .." in cmd
4) Write "make" in cmd
5) The plugin can now be found in the libs folder


It should be noted that absolute paths are used in this project. Therefore it is strongly adviced to either change the paths in the source files according or follow the instuctions in the readme.txt in the finalProject directory.
